# Gestion-Educativa-Comunas-Medellin.
Diseño de un sistema de información web orientado a fortalecer la gestión educativa en comunidades vulnerables de Medellín (Comunas 1, 3, 8 y 13)

Descripción del proyecto.  

El presente proyecto tiene como propósito diseñar un sistema de información web 
orientado a fortalecer la gestión educativa en comunidades vulnerables de 
Medellín, específicamente en las comunas 1, 3, 8 y 13, así como en algunos 
corregimientos. La propuesta surge a partir de la identificación de brechas 
educativas y digitales que limitan el acceso a recursos tecnológicos y 
oportunidades de aprendizaje en estos territorios.

La investigación se desarrolla bajo un enfoque mixto de tipo aplicado, integrando 
técnicas cualitativas y cuantitativas como encuestas, entrevistas y observación 
directa, con el fin de comprender las necesidades reales de la población. A partir 
de este diagnóstico, se plantea el diseño de un prototipo funcional que permita 
gestionar información académica, realizar seguimiento al progreso estudiantil y 
facilitar el acceso a recursos educativos digitales.

Este repositorio contiene el prototipo funcional desarrollado como evidencia del componente práctico de la Fase 4 del Proyecto de Grado.
