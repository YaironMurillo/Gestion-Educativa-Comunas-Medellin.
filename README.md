# Gestion-Educativa-Comunas-Medellin.
Diseño de un sistema de información web orientado a fortalecer la gestión educativa en comunidades vulnerables de Medellín (Comunas 1, 3, 8 y 13)

Fortalecimiento Educativo mediante TIC en Medellín.

Problema: Brechas digitales en sectores vulnerables que limitan el aprendizaje.

Tecnologías: Usaremos lenguajes como HTML, CSS, JavaScript y PHP/Python.

Equipo: Yairon Murillo, Yeli Pérez y Bladimir Pineda.
